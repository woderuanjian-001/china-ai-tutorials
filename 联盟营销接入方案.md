## 💡 为什么你这个站联盟营销 > AdSense？

| 维度 | AdSense | 联盟营销 |
|------|---------|---------|
| 用户意图 | 浏览（弱） | 注册/购买（强） |
| 单次转化收益 | $0.05-0.5（CPC） | $5-50（CPA） |

## 🛠️ 技术接入方案（在你的 Astro 站里落地）

### 方案 A：建立联盟链接统一管理（推荐）

不要把联盟链接硬编码到每篇文章里——以后换服务商就麻烦。建议建一个集中管理文件：`src/config.ts` 增加 `affiliate` 配置：

```typescript
// src/config.ts 追加内容
export const AFFILIATE_LINKS = {
  deepseek: {
    signup: "https://platform.deepseek.com/?ref=YOUR_CODE",
    label: "DeepSeek API Platform",
    disclosure: "We may earn credits if you sign up via this link.",
  },
  qwen: {
    signup: "https://www.aliyun.com/zh/referral?ref=YOUR_CODE",
    label: "Alibaba Cloud Bailian (Qwen)",
    disclosure: "We may earn commission if you sign up via this link.",
  },
  glm: {
    signup: "https://open.bigmodel.cn/pricing?ref=YOUR_CODE",
    label: "Zhipu GLM Platform",
    disclosure: "We may earn Tokens if you sign up via this link.",
  },
  kimi: {
    signup: "https://platform.kimi.ai",
    label: "Kimi API Platform",
    disclosure: "", // 暂无返佣
  },
  wildcard: {
    signup: "https://wildcard.com.cn/i/YOUR_CODE",
    label: "WildCard — Pay for Chinese AI APIs from abroad",
    disclosure: "We earn ¥100+ for each successful signup.",
  },
  novita: {
    signup: "https://novita.ai/?ref=YOUR_CODE",
    label: "Novita AI — One-stop API for Chinese models",
    disclosure: "We may earn credits if you sign up via this link.",
  },
} as const;
```

### 方案 B：建一个 AffiliateButton 组件

让联盟链接统一展示样式（含 FTC 合规声明）：

```astro
---
// src/components/AffiliateButton.astro
import { AFFILIATE_LINKS } from "../config";

export interface Props {
  platform: keyof typeof AFFILIATE_LINKS;
  text?: string;
  size?: "sm" | "md" | "lg";
}

const { platform, text, size = "md" } = Astro.props;
const link = AFFILIATE_LINKS[platform];
const btnText = text || `Get started with ${link.label}`;
---

<a
  href={link.signup}
  target="_blank"
  rel="noopener noreferrer sponsored"
  class={`affiliate-btn affiliate-btn-${size}`}
>
  {btnText} →
</a>
{link.disclosure && (
  <p class="affiliate-disclosure">⚠️ Affiliate disclosure: {link.disclosure}</p>
)}

<style>
  .affiliate-btn {
    display: inline-block;
    padding: 12px 24px;
    background: #d97706;
    color: #fff !important;
    border-radius: 8px;
    font-weight: 600;
    text-decoration: none;
    transition: filter 0.15s;
  }
  .affiliate-btn:hover { filter: brightness(1.1); }
  .affiliate-btn-sm { padding: 8px 16px; font-size: 0.875rem; }
  .affiliate-btn-lg { padding: 16px 32px; font-size: 1.125rem; }
  .affiliate-disclosure {
    font-size: 0.75rem;
    color: #6b7280;
    margin-top: 6px;
    font-style: italic;
  }
</style>
```

在文章里用一行调用：
```astro
<AffiliateButton platform="deepseek" />
```


## ⚠️ 重要合规与风险提示

### 1. FTC 披露必须做（美国法律要求）
- 每条联盟链接旁必须有「affiliate disclosure」
- 否则可能被 FTC 起诉（最高 $50,000 罚款）
- AffiliateButton 组件已内置披露

### 2. AdSense 与联盟营销并存规则
- AdSense 允许同页有联盟链接，但**不能让联盟链接伪装成广告**
- 联盟链接必须 `rel="sponsored"` 或 `rel="nofollow"`
- AffiliateButton 组件已用 `rel="noopener noreferrer sponsored"`

### 3. GitHub Pages 商用风险（再次提醒）
- GitHub Pages 条款：**「should not be used for commercial purposes」**
- 联盟链接 + AdSense 都算商业用途
- 流量起来后强烈建议**迁移到独立域名 + 正经托管**（Cloudflare Pages / Vercel 都免费）

### 4. 平台返佣政策会变
- 火山引擎的邀请活动已有「下线」记录
- 阿里云限时活动每月可能变
- **每季度检查一次各平台政策**，更新 `AFFILIATE_LINKS`

### 5. 不要伪造体验
- 「我用了 DeepSeek，强烈推荐」——如果你没用过别这么写
- 推荐内容必须基于真实评测，否则 FTC 也管
